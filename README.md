# LLM JSON Extraction Project

## 📌 Overview
This project extracts structured JSON data from unstructured text using LLM.

## 🚀 Features
- Convert text to JSON
- Clean structured output
- Easy to use

## 🛠️ Tech Stack
- Python
- LLM (Llama 3.2)
- JSON

## 📊 Output Example
Input: "Name: Ram, Age: 21"  
Output:
{
  "name": "Ram",
  "age": 21
}
